## Установка и запуск на Windows
- git clone https://github.com/EgorRU/Cyber-Squad
- cd Cyber-Squad
- pip install -r requirements.txt
- py main.py
- во втором терминале polling.py

## Команды
- /start - начать пользоваться
- /info - Получить актуальную информацию

## Команды для админа

- /admin - панель админа
